# RealWorld

[conduit](https://demo.realworld.io/#/)のクローンサイトです。

apiは以下のリポジトリです。

[Conduit API](https://github.com/makoto00000/realworld_api)

## 起動方法

```shell
npm run dev
```

上記のコマンドで起動し、[https://localhost:3000]を開きます。
（サイトを表示するには、APIを起動しておく必要があります。）
